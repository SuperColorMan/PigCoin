# Pig Coin 

Eco is built primarily in Java, with a few other technologies.

# What is the Pig Coin?

Used for circulation in the trade of agricultural commodities generally.
 
# Tokens address in Bsc

[Tokens Address](https://bscscan.com/token/0x3b015e51badaa07ad002b837f09bdaaf897824f1#balances)

# Distribution of tokens

* 20% : Initial airdrop, initial investment。
* 30% : Unlock the deal。
* 20% : Community reserve, generally used to reward outstanding contributions to the community。
* 18% : It is used to set up social public welfare funds for the care of vulnerable groups such as the elderly who live alone, women who have lost their only child, left-behind children, the disabled, poor families, unemployed people and so on。
* 12% : It will be used for the construction of offline physical farms, and 30% of the profits of offline farms will be used to buy back PigCoins。
